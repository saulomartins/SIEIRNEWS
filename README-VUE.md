# SIEIR Vue Dashboard (experimental)

Este diretório adiciona uma versão progressiva do dashboard usando Vue 3 + Vite e GridStack para painéis redimensionáveis e arrastáveis.

Quick start:

1. Instalar dependências:

```bash
npm install
```

2. Iniciar o servidor de desenvolvimento Vite:

```bash
npm run dev
```

3. Abra `http://localhost:5173` (porta padrão do Vite) no navegador.

Observação: Este é um scaffold inicial. Vou migrar os conteúdos de `public/dashboard-pro.html` e `public/js/dashboard-pro.js` para os componentes Vue em passos seguintes.
